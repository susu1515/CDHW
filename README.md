### Overview
This project develops a stage-resolved analytical framework to investigate global compound drought–heatwave (CDHW) events and their interactions with soil moisture from 1920–2019.

The codebase includes:

CDHW identification based on SPEI and Tmax thresholds

Detection of Time of Climate Impact Emergence (TCIE)

Calculation of Soil Available Moisture (ASM)

Identification of coupling-prone regions (ASM sensitivity regime)

Triggering-phase hydrothermal pathway diagnostics (ΔET–ΔT analysis)

Development-phase attribution using ExtraTrees + SHAP

Quantification of soil moisture depletion per unit CDHW intensity
